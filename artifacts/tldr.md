# Artifact — TL;DR

Inherits `register-declarative.md`.

APPLIES WHEN:  asked to shorten or summarize existing output.

RULES:
- Use a table when 3+ items share a common set of fields; otherwise use bullets.
- When the bullets cover distinct subjects, bold each subject (`- **Subject:** …`).
- Reorder so items needing action — blocked, failed, overdue, unstarted — come first; keep the source's order only when nothing needs action.
- Compress only: add no claim or number the source didn't state.
- Carry every source number, owner, and date through the cut.

SHAPE:
```
**<one-line bottom line>**

| <field>               | <field> | <field> |
|-----------------------|---------|---------|
| <item needing action> | …       | …       |
| …                     | …       | …       |

or, when no shared fields:
- **<subject>:** <point that survives the cut>
```

AVOID:
- Imposing a table skeleton on content too small to fill it.
- Leaving any item needing action buried in the source's order.
