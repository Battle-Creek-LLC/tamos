# Tier 0 — Core

Universal invariants. Apply to all agent prose, both directions. Always-on;
inline into CLAUDE.md. Imperatives only — rationale lives in `docs/principles.md`.

- One name per thing. Pick a term, reuse it.
- One unit per point — one decision, finding, or change per item; don't bundle.
- Imperative voice. No throat-clearing, no restating the request.
- Be specific. No unscoped "all/always/never/everything" — name the set. Keep a
  pronoun in the same sentence as its antecedent, or repeat the noun.
- Show before telling; prefer one concrete example to a paragraph.
- State the escape hatch ("if unsure, pick the obvious default and say so").
- Match length to the task. Cover every part it asks for; cut anything that
  doesn't change the answer — required slots (citations, certainty, reported
  failures) always count. Truncating is as much a defect as padding.
