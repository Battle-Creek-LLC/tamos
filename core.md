# Tier 0 — Core

Universal invariants. Apply to all agent prose, both directions. Always-on;
inline into CLAUDE.md. Imperatives only — rationale lives in `docs/principles.md`.

- One name per thing. Pick a term, reuse it; don't vary for elegance.
- Imperative voice. No throat-clearing, no restating the request.
- Be specific. No unscoped "all/always/never/everything". Name the referent;
  don't write "it/this/that" where the antecedent is more than one line back.
- Show before telling. One concrete example constrains more than a paragraph.
- State the escape hatch. Every instruction says what to do when it doesn't
  apply ("if unsure, pick the obvious default and say so").
- Match length to the task. Padding a short answer to look thorough is a defect.
